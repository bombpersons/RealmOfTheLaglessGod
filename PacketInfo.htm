© 2012 Anoos: 3

Reverse engineering the protocol game Realm Of The Mad God. / Build ver. 123 (?) /

Server Address: ec2-176-34-240-11.eu-est-1.compute.amazonaws.com (176.34.240.11) / EuEast /

The client requests a file from the server crossdomain.xml (http://www.adobe.com..._file_spec.html)

Information on packages == ==
The format of the package:
[LENGTH] <ID> (DATA)
| 01020304 | 05 | 0xDATA |

Presentation:
int length # length includes 5 bytes of header (4 bytes in length and one byte for ID), hence the length of the data
# == Length-5
ubyte id # ID package
Vector <byte> data # data is encrypted RC4, uses two keys:
# 311f80691451c71b09a13a2a6e -> Outgoing data stream <<<<MAY BE OLD KEYS>>>>
# 72c5583cafb6818995cbd74b80 <- input stream <<<<MAY BE OLD KEYS>>>>
List ========= package =======================
Presented in order of == as the developers ==
# Srv - the server sends a packet
# Cli - the client sends a packet
#?? - Assign the package is unknown


------- ------------ Format
ID NAME
type field
type2 field2
---------------------------
================================================

0 FAILURE # Srv, an error message
int errorID # ID errors
String errorDescription # description of the error

59 CREATE_SUCCESS # Srv, object creation was successful
int objectID # ID of the object
int charID # ID Team

49 CREATE # Cli, create a game object
short objectType # ID of the object type

20 PLAYERSHOO T # Cli, shoot
int timestamp # time shot
int bulletID # ID texture bullets
short containerType #???
Point startingPos # x, y coordinates start shooting
float angle # degrees angle of the shot in the plane xOy

61 MOVE # Cli, traffic Team
int tickID # ID game timer tick (anti-spidhak?)
int time # Timestamp the beginning of movement (anti-spidhak?)
Point newPosition # x, y coordinates of the destination
short recordsLength #???
Vector records #???

39 PLAYERTEXT # Cli, send a message to chat
String message # UTF-8 string posts

7 TEXT # Srv, a message from the chat
String name # The name of sender (player or bot that quest)
int objectID # ID of the object (?)
int numStars # The number of asterisks wound (made of class quests) player, who wrote the message
ubyte bubbleTime # Time as a bubble message will stay on the player (in seconds?)
String recipient # name of the recipient (for personal messages)
String text # Text
String cleanText # (formatted?) Message

21 SHOOT (_-u5, _-FL) # Srv, shot
ubyte bulletID # ID texture shot
int ownerID # ID "owner" of the shot
short containerType #???
Point startingPos # initial coordinates x, y shot
float angle # degrees angle of the shot in the plane xOy
short damage # Units of damage that will cause the bullet in contact with

31 SHOOT # Srv, handling multiple shots at once
ubyte bulletID # ID texture shot
int ownerID # ID "owner" of the shot
short containerType #???
Point startingPos # initial coordinates x, y shot
float angle # degrees angle of the shot in the plane xOy
short damage # Units of damage that will cause the bullet in contact with
# - Following is a list of shots, if there are bytes, then:
# Each following byte is the number of shots
# The following is a float followed by the angle that will be fired
# If no bytes, the number of shots = 1, and the angle = 0

38 DAMAGE # Srv, warning of damage caused and the effects (blindness, weakness, slowing, etc.)
int targetID # ID purposes
ubyte length # length of the list of effects
Vector effects # An array of bytes, ID effects (blind, slow)
int damageAmount # How much damage inflicted
bool kill # Killed?
ubyte bulletID # ID bullet
int objectID # ID of the object (?)

5 UPDATE # Srv, renewal of the world (new chunks (tiles), objects (monsters, players?), Drops (bags))
short length_tiles # length of the list tiles
Vector <_-MV> tiles # blocks to update the list (you need to understand the structure of the object block)
short length_newObjs # length of the list newObjs
Vector <_-Oe> newObjs # List of objects to be updated (----- *)
short drops_length # length of the list drops
Vector <int> drops # The list drops (id drops?)

14??

63 NOTIFICATI ON # Srv, notification
int objectID
String text # text alerts
int color # Color of text

57 NEW_TICK # Srv,?
int tickID
int tickTime # unix timestamp (?)
short length # length of the subsequent list
Vector <ObjectStatusData> statuses # The list of statuses (?)

65 INVSWAP # Cli, rearrangement of items in inventory
int time # timestamp permutations (for synchronization?)
Point position # Location (coordinates x, y) permutations (for drops?)
Slot slotObject1 # source object slots (slots for the need to define a data structure Slot)
Slot slotObject2 # The target slot (- ^)

58 USEITEM # Cli, the use of things (elixir, egg)
int time # timestamp use (sync?)
Slot slotObject # object slot
Point itemUsePos # The place where the object is used (to use the pots out of the bag right?)

77 SHOW_EFFECT # Srv, show the effect of
ubyte effectType # type of effect (0-15)
int targetObjectID # ID purposes
Point pos1 #???
Point pos2 #???
int color # Color effect (?)

17 HELLO # Cli, login to the server package
String buildVersion # version build # 123
Int gameID # ID games (?)
String guid # Login
String password # Password
String secret # The Secret
int keyTime #???
short keyLength # key length (?)
Vector <byte> key # key
int? length # length of the array
An array of String utf8bytes # utf-8 bytes

# Login, password and the secret is encrypted using public key PEM.RSA
# The encrypted data is encoded Base64
# RSA public key is available and is located inside the swf-file
# ----- BEGIN PUBLIC KEY ----- \ n
#MIGfMA0CGSqGSIb3DQEBAQUAA4GNADCBiQKBgQDEmgEmQcgLd0mvWqL6AKmhzjJfZoAmZC0PUmG8K9CB1M168P00S3eU+TSL5aG8Mg3Tipvs02gC2veC10knRi7r...



50 GOTO # Srv, pemereschenie object
int objectID # ID of the object moving
Point pos # Kordinaty move

11 INVDROP # Cli, drop a thing of the slot
Slot slotObject # These objects from the slot

75 INVRESULT # Srv, the result of action with the inventory
int result # result code

4 RECONNECT # Srv, reconnect to the (different) server
String name
String host
int port
int gameID
int keyTime
short keyLength
Vector <byte> key

41 PING # Srv
int serial # Serial number of ping

27 PONG # Cli
int serial # sequence number received in the processing of PING, this package should be sent immediately after the PING

37 MAPINFO # Srv, information on the map
int width # width of the card (a nexus 255)
int height # length of the card (a nexus 255)
String name # name card (Nexus)
uint fp #???
int background # background ID cards (2 - blue by default)
bool allowplayertp # Allow player teleport to each other (eg prohibited in the castle Oryx)


34 LOAD # Cli, cara download
int charID # ID chara

3 PIC # Srv, download a picture (bitmap)
int height
int width
Vector <byte> byteArray # array of data images, size: (height * width) * 4

28 SETCONDITI ON # Cli, installation conditions (?)
byte conditionEffect # ID effect
float conditionDuration # Duration of effect (seconds?)

23 TELEPORT # Cli, a request for teleportation of an object
int objectID

45 USEPORTAL # Cli, use the portal (or hatch in the dungeon)
int objectID

47 DEATH # Srv, notification of death
int accountID
int charID
String killedBy

68 BUY # Cli, a request for purchase (from a store in the Nexus)
int objectID

10 BUYRESULT # Srv, the result of the purchase
int result # result code
String resultString # message (error or a successful purchase)

62 AOE # Srv, AoE (Area of Effect) damage as a bomb jellyfish.
Point position # Position of the center
float radius # radius of effect
ushort damage # Damage Done
ushort origType # Damage type (?) is probably a bomb, ladders, etc. Khilki

19 GROUNDDAMA GE # Cli, causing damage to the ground (units)
int time
Point position

18 PLAYERHIT # Cli, getting into the player
byte bulletID
int objectID

24 ENEMYHIT # Cli, hit the enemy
int time
byte bulletID
int targetID
bool kill

26 AOEACK # Cli
int time
Point position

64 SHOOTACK # Cli
int time

6 OTHERHIT # Cli
int time
byte bulletID
int objectID
int targetID

16 SQUAREHIT # Cli
int time
byte bulletID
int targetID

51 GOTOACK # Cli
int time

55 EDITACCOUN TLIST # Cli
int accountListID
bool add
int objectID

66 ACCOUNTLIS T # Srv
int id
short listLength
Vector <int> accountIDs

74 QUESTOBJID # Srv
int objectID

40 CHOOSENAME # Cli, name change
String name

67 NAMERESULT # Srv, the result of change of name
bool success
String errorText

76 CREATEGUIL D # Cli, creating guild (clan)
String name # The name of the clan

69 CREATEGUIL DRESULT # Srv, the result of a guild (clan)
bool success
String errorText

8 GUILDREMOVE # Cli, removal of the guild (clan)
String name

22 GUILDINVIT E # Cli, invite to the guild
String name # The name of the invited players

53 ALLYSHOOT # Srv, shots allies (other players)
ubyte bulletID
int ownerID
short containerType
float angle

48 REQUESTTRA DE # Cli, a request to trade a player
String name

36 TRADEREQUE STED # Srv, warning players that want to trade with him
String name

9 TRADESTART # Srv, start trading
short myItemsLength # length of the list of things that I'm willing to trade
Vector <Slot> myItems # list of things
String yourName # Name (whose?)
short yourItemsLength # length of the list of things
Vector <Slot> yourItems # list of things

12 CHANGETRAD E # Cli, change the trade, boolean list of selected slots
short offerLength # length of the list of marks
Vector <bool> offer # The list of marks true - the slot is selected, false - not selected

44 TRADECHANG ED # Srv, a player has changed its proposal
short offerLength # length of the list of marks of slots
Vector <bool> offer # The list of marks of slots

15 ACCEPTTRAD E # Cli, to trade
short myOfferLength
Vector <Slot> myOffer
short yourOfferLength
Vector <Slot> yourOffer

25 CANCELTRAD E # Cli, cancel the trade
# The data is not transmitted

42 TRADEDONE # Srv, trade in finished
int code # result code
String desription # description

A TRADEACCEPTED # Srv, accepted trade
short myOfferLength
Vector <Slot> myOffer
short yourOfferLength
Vector <Slot> yourOffer

46 CLIENTSTAT # Srv, player stats client
String name # name of the article
int value # value

30 CHECKCREDI TS # Cli, titles, like pressing a button on the main menu Credits
# The data is not transmitted

33 ESCAPE # Cli, teleport to the Nexus
# The data is not transmitted

56 FILE # Srv, download the file from the server (?)
String name
int filesize
# This is followed by an array of UTF-8 bytes in size filesize

35 INVITEDTOG UILD # Srv, notification that the player client was invited to the guild (clan)
int objectID # ID guild
int charID # ID players invited to the guild

52 JOINGUILD # Cli, to accept an invitation to the guild
String guildName # Guild Name

60 CHANGEGUIL DRANK # Cli, change the player's rank in the guild (guild for admins)
String name # Player Name
int guildRank # new rank

13 PLAYSOUND # Srv, play a sound
int ownerID
ubyte soundID
